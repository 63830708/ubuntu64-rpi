# 安装
### 在windows上使用win32diskimager写入
### 在linux或mac上直接使用dd命令刷入
### ** 通过树莓派现有的系统刷系统 **
#### example1 : `zcat ubuntu.img.gz | sudo dd bs=4M of=/dev/mmcblk0`
#### example2 : 
```
unxz --verbose ubuntu.img.xz
sudo ddrescue -D -d --force ubuntu.img /dev/mmcblk0
```

### ！！！以上说明仅适用于chainsx构建的系统。
